# Аналіз layout-template та порівняння з promo-global-hr

## Загальна інформація

**Layout-template** (sam-chill-ua) та **promo-global-hr** - це два промо-лендінги з однаковою механікою, але різним візуальним оформленням.

## Структура проєктів

### Layout-template (sam-chill-ua)

#### Основні секції:
1. **`.welcome`** - Вітальна секція
   - Зображення персонажа (pers-desc.png / pers-mob.png)
   - Дерева з боків (welcome__tree-left, welcome__tree-right)
   - Кнопки авторизації/участі
   - Дата та інформація про промо

2. **`.challange`** - Секція виклику/челенджу
   - **Картки (7 штук)** - по одній на кожен день тижня
     - `.challange__current` - контейнер з картками
     - `.challange__current-item` - окрема картка
     - Стани: `_active`, `_lock`, `_open`
   - **Прогрес-бар** - `.challange__progress-bar`
   - **Блок з блюром** - `.challange__blur` (для заблокованих карток)
   - **Таби днів** - `.challange__tabs` (Понеділок-Неділя)
   - **Інформація про депозит** - `.challange__deposit`
   - **Кнопка відкриття** - `.btn-open`
   - **Навігація** - `.challange__nav` (ліворуч/праворуч, точки)

3. **`.progress`** - Секція прогресу
   - 7 днів прогресу (`.progress__days-item`)
   - Зображення персонажа (`.progress__right`)

4. **`.prize`** - Секція призів
   - **Слайдер призів** - `.prize__slider`
   - **Слайди** - `.prize__slide` (iPhone, Laptop, AirPods, Watch, Gift, FS, Points)
   - **Навігація** - `.prize__nav` (ліворуч/праворуч, точки)

5. **`.table`** - Таблиця результатів
   - **Таби тижнів** - `.table__tabs-item` (1-4 тижні)
   - **Заголовок таблиці** - `.table__head`
   - **Тіло таблиці** - `.table__body#table`, `#tableOther`

6. **`.games`** - Секція ігор
   - **Списки ігор по тижнях** - `.games__list._week1`, `._week2`, `._week3`, `._week4`
   - **Елементи ігор** - `.games__list-item`
   - **Кнопки** - `.games__btn.btn-casino`

7. **`.drop`** - Dropdown з правилами
   - `.drop--spins` - для спінів
   - `.dropdown` - деталі з правилами

8. **`.foot`** - Футер
   - Дерева з боків (`.foot__tree-left`, `.foot__tree-right`)
   - Фонове зображення

9. **`.popups`** - Попапи
   - Інформаційні попапи про призи
   - Виграшні попапи
   - Попап з таблицею призів

#### JavaScript функціональність:

1. **Визначення активного тижня**
   ```javascript
   const getActiveWeek = (promoStartDate, weekDuration) => {
       // Розрахунок активного тижня на основі дати старту промо
   }
   ```

2. **Система спінів**
   - `initSpin()` - ініціалізація спіну
   - `sendSpinRequest()` - відправка запиту на спін
   - Відкриття карток через API

3. **Прогрес користувача**
   - `setUserProgress()` - встановлення прогресу
   - `setStreakDays()` - встановлення streak днів
   - `displayBetsHistory()` - відображення історії ставок

4. **Слайдер призів**
   - Навігація ліворуч/праворуч
   - Точки навігації
   - Drag & drop підтримка

5. **Таблиця результатів**
   - Перемикання між тижнями
   - Завантаження даних з API
   - Відображення рейтингу

6. **Інтернаціоналізація**
   - Підтримка української та англійської мов
   - `translateKey()` - функція перекладу

### Promo-global-hr (поточний проєкт)

#### Основні секції:
1. **`.welcome`** - Вітальна секція
   - **Декоративні елементи** - `.welcome__decor-item` (gamepad, chip, coin, ball, cup, cube, globe)
   - **Заголовок** - `.welcome__title` (зображення)
   - **Кнопки** - авторизація/участь
   - **Призовий фонд** - `.welcome__prize` (40.000 €)
   - **Прогрес-бар** - Canvas з анімацією заповнення
   - **Текст** - інформація про промо

2. **`.participate`** - Секція участі
   - **Canvas анімація** - `#participateCanvas`
     - Олені з анімацією ходьби
     - Ялинка
     - Інтерактивність (клік по оленях → petting)
     - Сердечка при petting
   - **Dropdown** - деталі участі

3. **`.results`** - Секція результатів
   - **Таблиця** - `.table`
   - **Таби тижнів** - `.table__tabs-item` (1-4 тижні)
   - **Заголовок таблиці** - `.table__head`
   - **Тіло таблиці** - `.table__body`

4. **`.games`** - Секція ігор
   - **Списки ігор по тижнях** - `.games__list._week1`, `._week2`, `._week3`, `._week4`
   - **Елементи ігор** - `.games__item`
   - **Кнопки** - `.games__btn` (casino, sport)

5. **`.sticky-frame`** - Фрейм "телевізор"
   - Фіксована висота (700px десктоп, 550px мобілки)
   - Маска з рамкою (frame.png, frame-tab.png, frame-mob.png)

6. **Canvas сніжинки** - `#snowflakesCanvas`
   - 400 сніжинок
   - Початкове заповнення канвасу

#### JavaScript функціональність:

1. **Визначення активного тижня**
   ```javascript
   const getActiveWeek = (promoStartDate, weekDuration) => {
       // Аналогічна логіка до layout-template
   }
   ```

2. **Canvas анімація оленів**
   - `initParticipateCanvas()` - ініціалізація canvas
   - `Deer` клас - логіка оленів
   - Стани: `WALKING`, `IDLE`, `PETTING`
   - Анімація спрайтів
   - Інтерактивність (клік → petting)

3. **Прогрес-бар на Canvas**
   - `initProgressBar()` - ініціалізація
   - Анімація заповнення 1-100%
   - Послідовне з'явлення іконок

4. **Сніжинки**
   - `initSnowflakesCanvas()` - ініціалізація
   - 400 сніжинок
   - Початкове заповнення канвасу

5. **Адаптивність**
   - Фіксована висота для ефекту "телевізор"
   - Адаптивні розміри canvas
   - Масштабування оленів на мобільних

## Порівняння механіки

### Спільні елементи:

1. **Система тижнів**
   - Обидва проєкти використовують 4 тижні
   - Автоматичне визначення активного тижня
   - Перемикання списків ігор по тижнях

2. **Таблиця результатів**
   - Структура таблиці однакова
   - Таби для перемикання між тижнями
   - Завантаження даних з API

3. **Секція ігор**
   - 10 ігор на тиждень
   - 4 тижні = 40 ігор загалом
   - Структура однакова

4. **Авторизація/Участь**
   - Кнопки авторизації
   - Кнопки участі
   - Кнопки переходу в казино

5. **Інтернаціоналізація**
   - Використання `data-translate`
   - Підтримка кількох мов

### Відмінності:

#### Layout-template:
- **Система карток** - 7 карток для днів тижня
- **Система спінів** - відкриття карток через API
- **Прогрес по днях** - 7 днів streak
- **Слайдер призів** - горизонтальний слайдер
- **Візуал**: Теплі кольори (#F3B37A фон), персонаж, дерева

#### Promo-global-hr:
- **Canvas анімація** - олені з інтерактивністю
- **Прогрес-бар на Canvas** - анімація заповнення
- **Ефект "телевізор"** - фіксована висота з рамкою
- **Сніжинки** - фонова анімація
- **Візуал**: Холодні кольори (#000055 фон), pixel-art стиль, ретро естетика

## Візуальні відмінності

### Layout-template:
- **Кольорова схема**: Тепла (помаранчевий #F3B37A, рожевий #C776A0)
- **Стиль**: М'який, органічний
- **Елементи**: Дерева, персонаж, картки
- **Типографіка**: SuisseIntl, FavbetText, YanoneKaffeesatz

### Promo-global-hr:
- **Кольорова схема**: Холодна (синій #000055, зелений #05C48A)
- **Стиль**: Pixel-art, ретро, "телевізор"
- **Елементи**: Олені, ялинка, декоративні іконки
- **Типографіка**: PressStart2P, RobotoCondensed

## Технічні відмінності

### Layout-template:
- **API**: `https://fav-prom.com/api_sam_ua`
- **Структура**: Стандартна HTML структура
- **Анімації**: CSS transitions, слайдери
- **Висота**: Динамічна (scroll)

### Promo-global-hr:
- **API**: (не вказано в коді, але структура готова)
- **Структура**: Canvas для анімацій
- **Анімації**: Canvas API, requestAnimationFrame
- **Висота**: Фіксована (700px/550px) для ефекту "телевізор"
